Weather Dashboard

A simple web app that shows current weather information for your location. Built with React and Open-Meteo API.

Features

Detects your location automatically.

Shows current temperature, humidity, and wind speed.

Allows searching for weather in other locations.

Responsive and clean interface.

Installation

Clone the repository:

git clone https://github.com/yourusername/weather-dashboard.git
cd weather-dashboard


Install dependencies:

npm install

Run the app:

npm run dev

Usage

Allow the browser to access your location.

View weather data for your current location.

Search for other cities if needed.

Technologies

React – UI library

Axios – HTTP requests

Open-Meteo API – Weather data

Tailwind CSS – Styling
